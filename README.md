# 项目名称

MY_SCHEDULE

基于 Spring boot 的个人日程管理工具

## 项目结构

- `src/main/java/` - 包含 Java 源代码
- `src/main/resources/` - 包含配置文件和静态资源
- `src/test/java/` - 包含测试代码
- `pom.xml` - Maven 构建配置文件

## 环境要求

- JDK 版本：17
- Maven 版本：3.3.4
- 操作系统：Windows / Linux
- 日志路径：/home/logs
- 支持 Docker
